jQuery Plugins

  • Validation

  • The validation plugin allows for easy validation of forms. It makes the process extremely fast and resonsive.

  • FitText

  • FitText allows for expandable headings, and it makes it look and feel much more responsive. It also just so happens to work nicely in conjunction with the next plugin.

  • Lettering

  • Lettering allows for individually styling any given characters or words that you wish. It breaks them up into smaller groupings and makes them easier to style with CSS.

  • Canvas

  • Canvas is pretty amazing. It looks like there's a ton you can do with it, and the interactivity factor seems like it would be great for grabbing someone's attention. Reading data can be extremely boring, painting a picture with data is another story

  • DateDropper

  • DateDropper looks like a great way to get somebody to enter in a date, in not only a good looking way, but also very customizable way. It's versatile and can be used in different languages as well as for credit cards.

  • Tabs

  • Converted the exercise into a plugin, allowing for a dynamic tabular panel

  • JSON

  • Loaded JSON to display some pretty awesome inventors

  • AJAX

  • Used the xhr object to open up a XMLHttp request and then loaded the AJAX data (HTML)

  • Map

  • Used Google's library for displaying maps by getting a customized style, having the appropriate key and having the correctly formatted jQuery/JSON
